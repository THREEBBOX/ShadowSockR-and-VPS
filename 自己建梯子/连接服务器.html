连接服务器

如果你本地电脑使用的是 Windows 系统，那我推荐你下载一个叫 Git Bash 的软件来远程连接服务器 官方免费下载地址：https://git-scm.com/downloads Imgur 如果你是 Linux 或者 MacOS 用户，请忽略上面的软件

直接打开终端（Terminal），输入 ssh root@ip 其中 ip 替换成你服务器的 IPv4 地址, 按回车键，然后复制粘贴密码，按回车键即可登录。

粘贴密码时不显示密码，但不影响。 接着打开 git bash 软件，输入 ssh root@你的服务器ip地址 出现下面这张图，就说明你已经成功登陆你的服务器了！
